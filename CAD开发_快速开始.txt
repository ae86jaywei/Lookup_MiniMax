🚀 立即开始步骤:

第一步: 创建Visual Studio项目
1. 打开Visual Studio 2019+
2. 创建新的"类库"项目
3. 选择".NET Framework 4.8"
4. 项目名称: "ZWDynLookup"

第二步: 添加引用
在项目中添加对以下DLL的引用:
- ZwSoft.ZwCAD.dll (中望CAD API)
- ZwSoft.ZwCAD.DatabaseServices.dll
- ZwSoft.ZwCAD.Runtime.dll

第三步: 创建文件夹结构
在项目中创建以下文件夹:
- Commands/ (存放命令类)
- Models/ (存放数据模型)
- Service/ (存放服务类)
- UI/ (存放界面类)

第四步: 复制代码文件
从"核心文件内容.txt"中复制对应的代码到相应文件:
- PluginEntry.cs → 项目根目录
- BParameterCommand.cs → Commands/文件夹
- BActionToolCommand.cs → Commands/文件夹
- 其他文件根据注释提示放置

第五步: 编译和测试
1. 编译项目生成.dll文件
2. 在中望CAD中输入: APPLOAD
3. 选择编译生成的.dll文件
4. 插件加载成功后显示欢迎信息

🎮 使用插件功能:

基本命令:
- ZWBPARAMETER → 创建查寻参数
- ZWBACTIONTOOL → 创建查寻动作
- ZWBLOOKUPTABLE → 管理查寻表
- ZWBPROPERTIES → 管理参数特性

工作流程 (按照您的4步查寻功能流程):
步骤1: 使用其他类型参数命令创建基础参数
步骤2: 运行 ZWBPARAMETER 创建查寻参数
步骤3: 运行 ZWBACTIONTOOL 创建查寻动作
步骤4: 运行时使用查寻菜单进行参数切换
